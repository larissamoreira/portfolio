Be the Hero

Web - React.js

Mobile - React Native

This is an app to connect people and NGOs. The NGOs can signup on the app, and register their cases, with a title, description and the cost. People can check the cases out and contact the NGO through WhatsApp or Email to help on a case of their choice with a donation.

The API was made with Node.js, Frontend with React.js and Mobile with React Native. Code in Github
